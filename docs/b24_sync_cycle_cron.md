# Автосинк Б24 по расписанию

Эндпоинт цикла:

`/api/bitrix/sync_cycle.php?chunk=30`

Что делает 1 запуск:
- отправляет в Б24 очередной кусок остатков (storeproduct),
- отправляет в Б24 очередной кусок цен,
- читает очередной кусок товаров из Б24 и фиксирует расхождения.

## Рекомендуемый cron

Запуск каждые 3 минуты:

`*/3 * * * * /usr/bin/curl -s "https://your-domain/api/bitrix/sync_cycle.php?chunk=30" > /dev/null 2>&1`

Если нагрузка высокая, ставьте `chunk=15` и запуск каждую минуту.

## Где смотреть результат

- Вкладка `Интеграции`:
  - "Последний результат цикла"
  - "Найденные расхождения с Б24"
